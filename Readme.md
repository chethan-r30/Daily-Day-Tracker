## 📸 Overview

**DailyTracker** is a full-stack MERN web application built to help you take control of every hour of your day. From tracking daily habits to logging workouts with AI-powered coaching, studying smarter with subject targets, planning your week with a timetable, and learning something new every single day — DailyTracker has it all.

> 💡 Built with ❤️ by **Chethan** · Powered by **React**, **Node.js**, **MongoDB Atlas** & **OpenAI GPT-3.5**

## ⚡ Getting Started

### Prerequisites

Make sure you have installed:
- [Node.js](https://nodejs.org/) v18 or above
- npm v9 or above
- A [MongoDB Atlas](https://www.mongodb.com/atlas) account (free tier works)
- An [OpenAI API Key](https://platform.openai.com/) (optional — fallback content provided)

---

### 🔧 Installation

**1. Clone the repository**
```bash
git clone https://github.com/YOUR_USERNAME/daily-tracker.git
cd daily-tracker
```

**2. Install server dependencies**
```bash
cd server
npm install
```

**3. Install client dependencies**
```bash
cd ../client
npm install axios react-router-dom chart.js react-chartjs-2 react-toastify
```

---

### 🔑 Environment Setup

Create a `.env` file inside the `server/` folder:

```env
MONGO_URI=mongodb+srv://<username>:<password>@cluster.mongodb.net/dailytracker?retryWrites=true&w=majority
JWT_SECRET=your_super_secret_jwt_key_here
OPENAI_API_KEY=sk-your-openai-api-key-here
PORT=5000
```

> 💡 **MongoDB Atlas Setup:** Go to Atlas → Network Access → Add IP → Allow from anywhere (`0.0.0.0/0`)

---

### 🚀 Running the App

**Terminal 1 — Start the server:**
```bash
cd server
npm run dev
# ✅ Server running on http://localhost:5000
# ✅ MongoDB Connected
```

**Terminal 2 — Start the client:**
```bash
cd client
npm start
# ✅ React app running on http://localhost:3000
```

Open your browser and visit → **http://localhost:3000**

## 🌐 API Endpoints

| Method | Endpoint | Description | Auth |
|--------|----------|-------------|------|
| `POST` | `/api/auth/register` | Register new user | ❌ |
| `POST` | `/api/auth/login` | Login user | ❌ |
| `GET` | `/api/auth/profile` | Get user profile | ✅ |
| `GET` | `/api/activities` | Get all activities | ✅ |
| `POST` | `/api/activities` | Create activity | ✅ |
| `GET` | `/api/activities/logs/:date` | Get daily logs | ✅ |
| `GET` | `/api/activities/history` | Activity history | ✅ |
| `GET` | `/api/workouts/logs/:date` | Get workouts by date | ✅ |
| `POST` | `/api/workouts/logs` | Add workout + AI tip | ✅ |
| `POST` | `/api/workouts/bmi` | Calculate BMI | ✅ |
| `GET` | `/api/study/subjects` | Get all subjects | ✅ |
| `GET` | `/api/study/logs/:date` | Get study logs | ✅ |
| `GET` | `/api/timetable` | Get weekly timetable | ✅ |
| `POST` | `/api/timetable/settings` | Save wake/sleep time | ✅ |
| `GET` | `/api/timetable/daily/:date` | Get daily timetable log | ✅ |
| `POST` | `/api/timetable/ai-suggestion` | AI time management tips | ✅ |
| `GET` | `/api/whatsnew` | Get daily content (AI) | ✅ |
